# Security Policy

## Supported Versions

Currently only the `main` branch (latest commit) receives security updates. We do not maintain legacy major versions at this time.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover any security vulnerabilities within this project, please report them directly to the repository owners/administrators via internal communication channels rather than creating a public issue.

Please provide:
1. A summary of the vulnerability.
2. Steps to reproduce the issue.
3. Potential impact.

We take all security issues seriously and will respond promptly to address any confirmed vulnerabilities.
